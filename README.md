这是一个用于虚假医疗广告手法判别的数据库，主要用于 LlamaFactory 的监督微调（SFT）。

其中 1、2、3、4 这四个数据集都已经被我整理成可直接用于监督微调的格式。

1、2、4 这三个数据集来源于开源项目 CFAH（https://github.com/FuWeiru/CFAH），因此数据的具体内容请以他们的仓库为准；我只是做了整理与格式调整。

3 是我借助 AI 仿写扩展得到的数据集，共 349 条。

我将这些数据用于模型蒸馏：

以 data 为前缀的数据集：是在 1–4 号数据集基础上去除回答后的版本，便于直接喂给教师模型（只输入不含答案）。

以 teach 为前缀的数据集：是我将数据集输入 DeepSeek R1 671B 获取到的教师模型输出，作为教师数据使用。

This is a dataset for identifying deceptive tactics in fake medical advertisements, mainly used for supervised fine-tuning (SFT) with LlamaFactory.

Datasets 1, 2, 3, and 4 have all been reformatted so they can be directly used for supervised fine-tuning.

Datasets 1, 2, and 4 are derived from the open-source CFAH project (https://github.com/FuWeiru/CFAH
). For the exact original content, please refer to their repository; I only performed formatting, cleaning, and minor adjustments.

Dataset 3 was expanded via AI-assisted paraphrasing/imitation and contains 349 samples.

I use these datasets for model distillation:

Datasets prefixed with data: versions of datasets 1–4 with the answers removed, making them suitable to be fed directly into the teacher model (input-only format).

Datasets prefixed with teach: teacher outputs generated by feeding the datasets into DeepSeek R1 671B, and used as distillation teacher data.
